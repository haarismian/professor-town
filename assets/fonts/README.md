# Fonts Directory

Place the following WOFF2 font files in this directory:

1. **Source Serif 4** (700 weight)
   - `source-serif-4-bold.woff2`

2. **Inter** (400, 500, 600 weights)
   - `inter-regular.woff2`
   - `inter-medium.woff2`
   - `inter-semibold.woff2`

## Where to Get Fonts

- **Source Serif 4**: Available from Google Fonts or Adobe Fonts
- **Inter**: Available from Google Fonts or Rasmus Andersson's GitHub

For production use, ensure you have proper licensing for these fonts.
